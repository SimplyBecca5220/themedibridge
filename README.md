
# MediBridge AI: Health & Financial Inclusion

URL:https://themedibridge.lovable.app

MediBridge AI is a mobile-first platform designed to provide preliminary healthcare and financial resilience to underserved communities. It bridges the gap between high-tech medical guidance and real-world accessibility.

##​Core Functionality

• ​AI Symptom Triage: A clinical logic engine that categorizes symptoms into Low, Elevated, or Emergency risks, providing immediate "next-step" guidance.

• ​Medical Savings Ledger: A persistent "health wallet" integrated with Supabase that allows unbanked users to track micro-contributions for medical emergencies.

• ​Interactive Literacy: Action-oriented educational modules (Clean Water, Maternal Health, Malaria) that track user progress and reward completion with digital badges.

• ​Inclusive Design: A high-contrast, low-bandwidth UI that ensures the app remains functional in areas with limited connectivity.

##​Tech Stack & Proof of Work

​Built with React, Tailwind CSS, and Supabase, the app features a fully functional database schema for tracking triage history and financial transactions. It demonstrates a scalable solution for Pillars 1, 2, and 4 by turning static health information into a dynamic, data-driven support system
